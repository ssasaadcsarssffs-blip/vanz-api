Base URL

Semua request harus diarahkan ke URL dasar berikut:

https://vanz-api-one.vercel.app

Daftar Endpoints

Silakan tes request secara live pada masing-masing endpoint di bawah ini.

GET /api/ai/vanz-ai

Berinteraksi dengan model VANZ AI menggunakan parameter prompt teks untuk menghasilkan jawaban responsif.

https://vanz-api-one.vercel.app/api/ai/vanz-ai?prompt=Siapa%20kamu%3F
Response Result
GET /api/canvas/fakenotif

Membuat gambar notifikasi tiruan bergaya iOS Lockscreen berdasarkan parameter teks dan URL foto profil.

https://vanz-api-one.vercel.app/api/canvas/fakenotif?ppurl=https%3A%2F%2Fuploader.zenzxz.dpdns.org%2Fuploads%2F1783937277449.jpeg&username=ZennzXD&chat=Haii%20sayangg%20udah%20makan%20belumm%3F%20%F0%9F%A5%BA&tanggal=Senin,%206%20Maret&jam=18.30
Response Result
GET /api/maker/brat

Membuat gambar bergaya Brat kustom menggunakan parameter input teks.

https://vanz-api-one.vercel.app/api/maker/brat?text=VanzDev
Response Result
GET /api/downloader/allinone

Mengunduh media (audio/video) dari berbagai platform seperti TikTok, YouTube, IG, dll. dengan opsi format MP3 atau MP4.

https://vanz-api-one.vercel.app/api/downloader/allinone?url=https%3A%2F%2Fvt.tiktok.com%2FZSXXtFL7c&format=mp3
Response Result
GET /api/tools/removebg

Menghapus latar belakang (background) dari gambar berdasarkan URL foto yang dikirimkan.

https://vanz-api-one.vercel.app/api/tools/removebg?image=https%3A%2F%2Fuploader.zenzxz.dpdns.org%2Fuploads%2F1783937277449.jpeg
Response Result
