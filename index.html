Selamat Datang di Vanz Api

Platform REST API gratis, cepat, dan stabil untuk kebutuhan pengembangan aplikasi dan bot Anda. Dibuat khusus dengan performa tinggi.

Kenapa memilih Vanz Api?

⚡ **Respon Cepat:** Server dioptimasi penuh tanpa hambatan delay.
🔒 **Aman:** Menggunakan enkripsi SSL modern.
🤖 **Mudah Digunakan:** Dokumentasi ringkas dan contoh kode siap pakai.

Dokumentasi Dasar

Pelajari cara mengintegrasikan Vanz Api ke dalam projek aplikasi Anda.

Base URL

Semua request harus diarahkan ke URL dasar berikut:

https://api.vanzdev.xyz/v1

Contoh Response (JSON)

{
  "status": 200,
  "creator": "Vanz Dev",
  "result": {
    "message": "Halo Dunia! Berhasil terhubung ke Vanz Api."
  }
}

Daftar Endpoints

Berikut adalah daftar fitur API yang tersedia dan siap Anda gunakan.

GET /downloader/tiktok

Mengambil data video Tiktok tanpa watermark lewat URL.

Parameter: ?url=LINK_TIKTOK

GET /tools/ai

Berinteraksi dengan kecerdasan buatan cerdas.

Parameter: ?query=PERTANYAAN
