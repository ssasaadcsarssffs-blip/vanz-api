Dokumentasi Dasar

Pelajari cara mengintegrasikan Vanz Api ke dalam projek aplikasi Anda.

Base URL

Semua request harus diarahkan ke URL dasar berikut:

https://vanz-api-one.vercel.app

Contoh Response (JSON)

{
  "status": 200,
  "creator": "Vanz Dev",
  "result": {
    "message": "Halo Dunia! Berhasil terhubung ke Vanz Api."
  }
}

Daftar Endpoints

Berikut adalah daftar fitur API yang tersedia dan siap Anda gunakan.

GET /api/ai/claude

Berinteraksi dengan Artificial Intelligence model Claude untuk menjawab berbagai macam pertanyaan teks secara cerdas dan kontekstual.

Query Parameters:

  • text (Required) : Pertanyaan atau instruksi yang ingin dikirim ke AI.
  • apikey (Optional) : Kunci akses API milik pengguna (jika diperlukan).

Contoh Request:

https://vanz-api-one.vercel.app/api/ai/claude?text=siapa penemu lampu?
